ダイスセットの操作

サイドバーで複数のダイス設定を管理できます。

ダイスセットの使い方​

ダイスセットをメインエリアで使用する方法は、目的に応じて2通りあります。

ダッシュボードで使う​

サイドバーからダイスセットをダッシュボードへドラッグ＆ドロップします。

  • 単一セット: 1つのダイスセットをドロップすると、そのセットの結果がグラフに表示されます
  • 複数セット（比較）: 複数のダイスセットを選択状態（アクティブ）にしてまとめてドロップすると、結果が重ねて表示され比較分析できます。各セットは異なる色で表示されます
複数選択の方法

サイドバーのリストアイテムをクリックすると選択状態（アクティブ）になります。複数のセットをクリックして選択してから、まとめてドラッグ＆ドロップしてください。

シート（結果一覧）で使う​

サイドバーからダイスセットをシート（結果一覧）タブへドラッグ＆ドロップします。

ロールを実行すると、表形式で個々の試行結果を確認できます。

注記

シートでは単一のダイスセットのみを表示・分析できます。

編集​

編集ボタン（）またはメニュー（︙）の「編集」から、ダイス式や設定を変更できます。

編集可能な項目​

  • セット名: ダイスセットの識別用の名前
  • ダイス式: ダイスの個数、面数、修正値（例: 2d6+3）
    • ダイス数: 1〜10個
    • 面数: 1〜9999面
    • 修正値: 正の数（加算）または負の数（減算）、最大±99999
  • 試行回数: シミュレーション実行時の振り直し回数（1〜1,000回）

編集の手順​

  1. サイドバーのダイスセットで、編集ボタン（）をクリック、またはメニュー（︙）から「編集」を選択
  2. 編集ダイアログが開くので、必要な項目を変更
  3. 「保存」をクリックして変更を確定
ヒント

編集内容は即座に保存され、次回の計算に反映されます。過去の結果には影響しません。

複製/削除​

メニュー（︙）から、セットの複製や削除が行えます。

制限事項​

  • 最大20個まで保存可能です
  • 一度に振れるダイスの最大数: 10個
  • 一回あたりの最大試行回数: 1,000回
対応しているダイス記法

現在は「ダイス数」「面数」「修正値」を用いた、基本的な加算・減算ロールに対応しています（例: 2d6+3, 1d100-5）。
