Get Started Features

Main Features

Dice Simulator is packed with features to make your TRPG sessions more fun and convenient

Dashboard

Grasp dice statistics at a glance. Visually check dice roll trends in real-time.

Dice Set Management

Save frequently used dice combinations. Switch between multiple dice sets for efficient session management.

Detailed Analysis

Analyze dice distribution and trends in depth. Check statistical data filtered by specific conditions.

Customizable

Freely configure display items and themes. Customize the interface to match your play style.

Filter Functionality

Filter by dice type or period. Extract only the data you need for efficient analysis.

Responsive Design

Compatible with PC, tablet, and smartphone. Use comfortably from any device.
